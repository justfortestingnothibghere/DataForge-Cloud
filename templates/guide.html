1. Signup & Get API Key

Visit Signup or use API:

curl -X POST https://yourdomain.com/auth/signup \
-H "Content-Type: application/json" \
-d '{"name":"John","username":"john","email":"john@example.com","password":"pass123"}'

Copy your API key from dashboard.

2. Upload Data

Via dashboard or API:

curl -X POST https://yourdomain.com/api/upload \
-H "Authorization: Bearer YOUR_API_KEY" \
-F "type=image" \
-F "file=@/path/to/photo.jpg" \
-F "share=true"

3. Access Your Data

GET with API key:

curl -X GET "https://yourdomain.com/api/v2/john?uploads=123" \
-H "Authorization: Bearer YOUR_API_KEY"

Response: 

{ "owner": "john", "type": "image", "file_url": "/uploads/123_john.png" }

For shared links: Use /api/share/{id}?token=...

4. Real-Time Logs

Connect via WebSocket:

const ws = new WebSocket('ws://yourdomain.com/ws/logs');
ws.onmessage = (e) => console.log(e.data);
